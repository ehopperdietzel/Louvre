Detailed Description

Representation of a client surface (usually refered as window).

A surface is the representation of a client window that can be drawn on screen. It has a position, size, texture, and a role.
Initially all surfaces have no role (Undefined), you must wait for them to get a role (notified with typeChangedRequest()) in order to be able to draw them on screen. A list of all created surfaces can be accessed from the LCompositor class, check the LCompositor::surfaces() method. A list of all surfaces created by a client can be accessed from the LClient class, check the LClient::surfaces() method.
